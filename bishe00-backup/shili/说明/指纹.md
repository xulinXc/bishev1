## 测试建议

### 容易找到的测试目标（公网常见）
1. **OA系统**: 使用 fofa 搜索 `app="泛微"` 或 `app="致远"` 或 `app="用友"`
2. **VPN系统**: 搜索 `app="深信服VPN"` 或 `app="Cisco VPN"`
3. **开发工具**: 搜索 `app="Jenkins"` 或 `app="Gitlab"` 或 `app="JIRA"`
4. **中间件**: 搜索 `app="Weblogic"` 或 `app="Tomcat"`
5. **监控系统**: 搜索 `app="Zabbix"` 或 `app="Grafana"`
6. **防火墙**: 搜索 `app="深信服"` 或 `app="天清汉马"`
7. **宝塔面板**: 搜索 `app="宝塔"`
8. **Swagger**: 搜索 `title="Swagger UI"`

### 特定指纹测试
- **Shiro**: 搜索响应头包含 `rememberMe=`
- **ecology**: 搜索响应头包含 `ecology_JSessionid`
- **列目录**: 搜索响应体包含 `Index of /`
