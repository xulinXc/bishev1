# 类图补充部分使用说明

## 概述

`类图-补充部分.puml` 是主类图（`类图.puml`）的补充，展示了主类图中被遮挡的部分。这两个类图配合使用可以展示完整的系统类结构。

## 包含的包

补充部分类图包含以下三个包：

1. **WAF绕过包** - 载荷变体类
2. **AI提供者包** - AI相关接口和实现类
3. **其他包** - 指纹规则、指纹数据、字典信息

## 与主类图的连接关系

补充部分类图中已经包含了与主类图的连接关系：

### 1. AI提供者与主类图的连接

- **基础聊天会话** → **AI提供者**：基础聊天会话使用AI提供者进行聊天
- **基础聊天会话** → **聊天消息**：基础聊天会话包含多条聊天消息
- **MCP连接** → **AI工具**：MCP连接提供AI工具列表

### 2. WAF绕过与主类图的连接

- **WAF绕过请求** → **载荷变体**：WAF绕过请求使用载荷变体进行测试

### 3. 其他类与主类图的连接

- **目录扫描请求** → **字典信息**：目录扫描请求使用字典信息进行扫描

## 使用方式

### 方式一：分别生成两个图片

1. 使用 PlantUML 生成 `类图.puml` 的图片
2. 使用 PlantUML 生成 `类图-补充部分.puml` 的图片
3. 在论文中并列放置两个图片，并标注说明

### 方式二：在论文中使用

在论文中可以这样描述：

> 由于系统类图较大，为了清晰展示，将类图分为两部分：
> - 图X-X：主类图（包含任务管理、扫描请求、POC规则、EXP验证、MCP接口等）
> - 图X-X：类图补充部分（包含AI提供者、WAF绕过、其他辅助类等）
> 
> 补充部分类图中的类通过以下关系与主类图连接：
> - 基础聊天会话使用AI提供者接口进行AI对话
> - WAF绕过请求使用载荷变体进行WAF绕过测试
> - 目录扫描请求使用字典信息进行目录扫描

## 生成图片命令

### 使用 PlantUML 命令行

```bash
# 生成主类图
plantuml 类图.puml

# 生成补充部分类图
plantuml 类图-补充部分.puml
```

### 使用在线工具

1. 访问 http://www.plantuml.com/plantuml/
2. 复制 `类图-补充部分.puml` 的内容
3. 粘贴到在线编辑器中
4. 点击生成图片

## 注意事项

1. 补充部分类图中标注了 `(主类图中)` 的类，表示这些类在主类图中已定义
2. 连接关系中的箭头方向表示依赖关系
3. 两个类图配合使用可以完整展示系统的类结构

## 类图说明

### AI提供者包

- **AI提供者接口**：定义了AI聊天的基本接口
- **聊天流提供者接口**：定义了流式聊天的接口
- **具体实现类**：OpenAI提供者、DeepSeek提供者、Anthropic提供者、Ollama提供者
- **辅助类**：聊天消息、AI工具、AI工具调用、流控制器

### WAF绕过包

- **载荷变体**：定义了WAF绕过测试的载荷变体

### 其他包

- **指纹规则**：定义了Web指纹识别规则
- **指纹数据**：包含多条指纹规则
- **字典信息**：定义了字典文件的元信息
